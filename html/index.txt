<h3>Examples</h3>

This is a simple set of utilities for drawing interactively in a MATLAB figure window. To test run

<p><code>&gt;&gt; imageLabel</code>

<p>Two windows will appear, one a drawing window with an image and a second with an edit control window. You can draw different shapes by selecting the relevant radio buttons. Objects can also be copied and moved. Selecting edit allows you to delete the selected object by pressing the delete key.
